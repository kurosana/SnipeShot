# ねらいうちゲーム データ更新手順

## 概要

ゲームのポケモンデータは [PokeAPI](https://pokeapi.co/) の CSV からビルドした JSON を使用しています。
新世代の追加やデータ修正は、PokeAPI の更新を取り込んで **build_db.bat** を実行するだけで反映できます。

## 必要なもの

- Python 3（`python` コマンドが使えること）
- PokeAPI リポジトリの CSV データ

## データの場所

| 項目 | パス |
|------|------|
| PokeAPI CSV（入力） | `..\pokeapi-master\pokeapi-master\data\v2\csv\` |
| 生成 JSON（出力） | `SnipeShot\data\` |
| ビルドスクリプト | `SnipeShot\scripts\build_data.py` |

## 更新手順

1. [PokeAPI/pokeapi](https://github.com/PokeAPI/pokeapi) から最新版を取得する
2. 次のフォルダの中身を、手元の CSV フォルダに**上書き**する  
   `C:\Users\write\Root\ToolMaker\pokeapi-master\pokeapi-master\data\v2\csv\`
3. `SnipeShot` フォルダで **build_db.bat** をダブルクリック（またはコマンドプロンプトから実行）
4. コンソールに `OK mode_*.json` と表示されれば成功
5. ブラウザでゲームを開き直して動作確認する

## データ統合ルール

- **タイプ・特性・種族値**: そのモードの最大世代までの最新データを使用
- **覚える技**: 対象ゲームのいずれかで覚えれば OK（OR 統合）。進化後は進化前の技をすべて引き継ぐ（進化前に進化後の技は含めない）
- **対象ポケモン**: 各モードのゲームに登場するポケモン（デフォルト形態を基本とする）
- **メガシンカ・フォルムチェンジ**: デフォルト形態と覚える技が異なる形態は別エントリとして追加。覚える技が同じ形態はデフォルトに統合
- **進化・フォルムラインのカウント**: 同一進化チェーン（`evolution_chain_id`）で1グループ。メガ/フォルムも同じラインに含める
- **一騎打ち判定**: 該当ポケモンに含まれる進化・フォルムラインの数が2のとき発動

## 表示・見た目の設定

ルートの `config.json` で以下を編集できます（ビルド不要）。

- `version` / `releaseNotes`: スタート画面左上のバージョン表示
- `appTitle`: タイトル
- `rulesHtml`: スタート画面のルール説明（HTML）
- `colors`: 画面の色合い（CSS変数）

## ローカルデバッグモード

プロジェクトルートに空ファイル `local.debug` を置くと（`.gitignore` 対象）、ゲーム画面バナーの「もう一度」横に **デバッグ** ボタンが表示されます。

- 1回目: 降参と同様に該当ポケモン一覧を表示
- 2回目: 「該当ポケモンは…」の集計表示に戻る（降参とは異なりトグル動作）

`start.bat` でローカル起動したときのみ使う想定です。GitHub Pages には `local.debug` を置きません。

## 自動で非表示になるモード

ビルド時に「技データが1件もない」世代は `index.json` から自動的に除外されます。
例: PokeAPI に ZA の学習技が未収録の間は「ZA」は選択肢に出ません。
データが追加されたあと build_db.bat を再実行すれば、自動的に選択肢に現れます。

## トラブルシューティング

### `CSV directory not found` と表示される

`pokeapi-master` のパスがずれています。`scripts/build_data.py` 内の `CSV_DIR` が  
`SnipeShot` の兄弟フォルダ `pokeapi-master\pokeapi-master\data\v2\csv` を指しているか確認してください。

### Python が見つからない

https://www.python.org/ から Python 3 をインストールし、インストール時に「Add to PATH」にチェックを入れてください。

### ゲームで古いデータが表示される

ブラウザのキャッシュを削除するか、スーパーリロード（Ctrl+Shift+R）してください。
